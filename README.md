# LendingClub-Loan-Default-Risk-Model
End-to-end credit-risk prediction project using SQL, Python (Machine Learning & Calibration), and Power BI dashboard.

## Data Source
The dataset used in this project is the **LendingClub Loan Data**, sourced from Kaggle.

- Source: https://www.kaggle.com/datasets/chiebukar/lending-club-loan-data
- Description: Historical LendingClub loan-level data including borrower attributes, loan characteristics, and default outcomes.
- Note: The raw dataset is not included in this repository due to size and licensing considerations.
